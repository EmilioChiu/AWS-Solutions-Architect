# AWS Solutions Architect

Este es un curso auto-guiado, diseñado para aprender AWS Solutions Architect desde cero con la ayuda de un mentor IA. El curso fue construido por **course-constructor** ([@EmilioChiu](https://github.com/EmilioChiu)) a partir del diseño generado por **course-designer** ([@nove2311na](https://github.com/nove2311na)), y está optimizado para usarse con **/teach** ([@mattpocock](https://github.com/mattpocock)) como mentor.

## Propósito

Obtener la certificación AWS Solutions Architect Associate (SAA-C03) con conocimientos prácticos aplicables como backend developer.

## Para quién es

**Nivel:** beginner (experiencia como backend developer web)
**Dedicación:** 7-10 horas por semana

## Mentor sugerido

Carga la skill **/teach** al iniciar una sesión de estudio.

## Contenido

- 10 semanas, 50 sesiones
- Cada sesión tiene su carpeta `week-N/day-M/practice/` para ejercicios
- Las lecciones se guardan en `lessons/`
- El progreso personal se registra en `progress.json`
- La deuda técnica se rastrea en `debt.json`

## Notas

- `debt.json`, `progress.json` y `NOTES.md` son personales y están excluidos de git
- `learning-records/`, `reference/`, `lessons/` y `GLOSSARY.md` se llenan durante el estudio
